# Vue 3 + Vite2 + phaser 学习官网 demo

- 基于[https://github.com/ningbnii/vite2_vue3_base](https://github.com/ningbnii/vite2_vue3_base)
- phaser 官网 demo(https://phaser.io/examples/v3)

## 安装测试

```
// 进入目录
yarn
// 启动server
yarn dev
// 打包
yarn build
```
